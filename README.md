# Terraform Learning and Practice

This repository contains code and resources for learning and practicing Terraform.

    ├── variables.tf
    ├── outputs.tf
    ├── modules/

    │   │   ├── variables.tf
    │   │   ├── outputs.tf
    │   │   └── main.tf
    │   ├── compute/

    │   │   ├── variables.tf
    │   │   ├── outputs.tf
    │   │   └── main.tf
    │   └── storage/

    │       ├── variables.tf
    │       ├── outputs.tf
    │       └── main.tf
    ├── providers/

    │   │   ├── main.tf
    │   │   ├── variables.tf
    │   │   └── main.tf
